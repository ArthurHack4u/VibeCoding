HTML

DEFINICIONES









































Etiquetas HTML

HTML
Lenguaje de marcas de hipertexto.
BODY
Contenedor del contenido principal de la página.
HEAD
Define el encabezado de un documento HTML.
TITLE
Especifica el título de la página, que se muestra en la pestaña del navegador.

Elementos Estructurales

HEADER
Define la cabecera de la página o sección.
FOOTER
Define el pie de página.
NAV
Define el área de navegación del documento.
MAIN
Define el contenido principal de la página.
